The voronoi diagram

Given a set of points V called Voronoi Points in a 2D region, the Voronoi diagram is a decomposition of that region in cells where each cell contains exactly one Voronoi point. For every point X in the cell, the closest voronoi point to X is the voronoi point associated with that cell.

To create this diagram We use the incredible awesome Fortuna Alghorithm. One of the most important achievments in Computational Geometry. This little application illustrates the main ideas behind the alghorithm. Also check the book Berg, Mark, Computation Geometry: Algorithms and Applications
